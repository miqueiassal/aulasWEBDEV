# Projeto WebMac


## Rede de avaliações de obras
 
Baseado em sites/app como MyanimeList e skoob. 

principalmente:

+ animes,

+ mangas, 

+ webtoons.

### Páginas previstas:

 + Página para login,
 
 + Páginas com suas obras vistas/lidas/ Esperadas

+ feed com avaliações de amigos.

### Funcionalidades previstas
 
 + Avaliação com três niveis: Gostei, neutro e Não gostei,

 + Adicionar novas obras,

 + Planejar próximas obras,

 + Avaliacão escrita,

 + Adicionar amigos.


#### Obs:

Esse é apenas um esboço, o projeto final sofrerá alterações.
